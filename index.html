// Core Capabilities

Built to Perform
Under Pressure

🎥
Live Webcam + RTSP
Connect any USB webcam or IP camera via RTSP URL. Real-time 30FPS inference with frame skipping for maximum performance.
⚡
GPU Accelerated
Auto-detects CUDA GPU. Half-precision (FP16) inference on NVIDIA cards. CPU fallback when GPU is unavailable.
🧠
ResNet-18 Model
Fine-tuned ResNet-18 with ImageNet pre-training. TorchScript export for fast loading. 95%+ accuracy on validation set.
🚨
Smart Alert System
Debounced audio alerts on threat detection. Pulsing visual borders, animated badges, and real-time status indicators.
🎨
Premium UI
Glassmorphism dashboard with animated confidence bars, shimmer effects, pulsing indicators, and smooth micro-interactions.
🏋️
Built-in Trainer
Train directly from the GUI or CLI. Cosine LR scheduler, label smoothing, augmentation pipeline. Auto-loads best checkpoint.

FIRE
Detects flames, smoke, and fire incidents. Red pulsing border + audio alert triggers immediately.
⚠️
ACCIDENT
Identifies road accidents, collisions, and crash scenes. Orange warning with sound notification.
✅
NORMAL
Scene classified as safe. Green status indicator. No alerts. Continuous monitoring active.

// Quick Start

Up and Running
in 4 Commands

TERMINAL — VIGIL-88 SETUP

# 1. Clone the repository $ git clone https://github.com/YOUR_USERNAME/vigil-88.git $ cd vigil-88
# 2. Create virtual environment (Python 3.11) $ py -3.11 -m venv venv && venv\Scripts\activate
# 3. Install dependencies (GPU — CUDA 12.1) $ pip install torch==2.2.2 torchvision==0.17.2 --index-url https://download.pytorch.org/whl/cu121 $ pip install PyQt6 opencv-python==4.8.1.78 Pillow "numpy<2"
# 4. Train model + launch $ python train.py --dataset ./dataset --epochs 15 $ python main_app.py
✓ Model loaded on CUDA (NVIDIA GeForce RTX 3050 6GB Laptop GPU) ✓ AI ENGINE READY — 95.8% validation accuracy
